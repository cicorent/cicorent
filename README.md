# CICO Rent - Sito statico (HTML/CSS/JS)

Sito one-page moderno, minimale ed elegante per CICO Rent.

## Come usarlo
1) Apri `index.html` nel browser (doppio click).
2) Se lo vuoi online: carica l'intera cartella su qualsiasi hosting statico.

## Cosa modificare in 2 minuti
- **Social**: nel footer sostituisci i link `#` con gli URL reali di Instagram/Facebook/TikTok.
- **Immagini veicoli**: puoi rimpiazzare gli SVG in Hero con PNG trasparenti:
  - metti i file in `assets/` (es. `polo.png`, `boxer.png`)
  - sostituisci i blocchi `<svg>...</svg>` con `<img src="assets/polo.png" alt="Volkswagen Polo" />` e `<img src="assets/boxer.png" alt="Volkswagen Boxer" />`
